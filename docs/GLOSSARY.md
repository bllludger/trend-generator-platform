# Глоссарий терминов

Единая терминология продукта (пользовательский UI, админка, внутренние сущности). Подробнее про согласованность формулировок — [TERMINOLOGY_CONSISTENCY_REVIEW.md](TERMINOLOGY_CONSISTENCY_REVIEW.md).

---

## Бренд и продукт

| Термин | Значение |
|--------|----------|
| **NeoBanana / Neo Banana** | Бренд продукта. В пользовательских текстах и маркетинге — единообразно (например «NeoBanana Preview» на превью). |
| **Trend Generator** | Описание продукта: бот для генерации стилизованных фото по трендам. |

---

## Покупка и лимиты

| Термин | Значение |
|--------|----------|
| **Пакет** | Тариф/единица покупки: Trial, Neo Start, Neo Pro, Neo Unlimited. Пользователь «покупает пакет»; в админке — «Пакеты». |
| **Купить пакет** | Основной CTA в боте (меню «🛒 Купить пакет»). |
| **Сессия** | Технический термин: активная фотосессия после покупки пакета (таблица `sessions`). В пользовательском UI предпочтительно «пакет» или «остаток фото». |
| **Осталось фото** | Счётчик: сколько снимков осталось по текущему пакету (takes_used / takes_limit). |
| **4K** | Высокое разрешение без водяного знака. «Забрать в 4K», «4K без водяного знака». |
| **Водяной знак / watermark** | Превью с подписью. В русском UI — «водяной знак», «без водяного знака». |
| **HD** | В контексте доставки то же, что 4K: «доставка HD», «забрать в HD». |

## Генерация и контент

| Термин | Значение |
|--------|----------|
| **Тренд** | Визуальный стиль/сцена для генерации (например «Ретро», «Кино»). Пользователь «выбирает тренд». |
| **Тема (тематика)** | Категория трендов (например «Кино», «Арт»). Сначала выбор темы, затем тренда. |
| **Фото** | Результат генерации в пользовательском UI. В админке и коде допустимы «снимок», «Take» (сущность). |
| **Take** | Один «снимок» в контексте сессии: три варианта A/B/C, превью и оригиналы. Внутренняя сущность (таблица `takes`). |
| **Job** | Legacy-задача генерации одного изображения (без сессии). Внутренняя сущность. |
| **Превью** | Изображение с водяным знаком до оплаты/разблокировки. |
| **Избранное** | Выбранные пользователем варианты (по одному на снимок) для последующей доставки в 4K. |

## Платежи

| Термин | Значение |
|--------|----------|
| **Stars** | Telegram Stars — нативная оплата в боте. |
| **ЮMoney** | Оплата рублями (инвойс в чате или по ссылке через ЮKassa). |
| **ЮKassa** | Платёжная система для приёма оплаты по ссылке (unlock одного фото, покупка пакета). |
| **Unlock** | Разблокировка одного фото (убрать водяной знак) — по Stars, токенам, 4K-кредитам или по ссылке ЮKassa. |
| **Pack order** | Заказ покупки пакета по ссылке ЮKassa (таблица `pack_orders`). |
| **Unlock order** | Заказ разблокировки одного фото по ссылке ЮKassa (таблица `unlock_orders`). |

## Админка и внутренние термины

| Термин | Значение |
|--------|----------|
| **Пакеты (в админке)** | Раздел настройки тарифов. Заголовок: «Пакеты» или «Пакеты (цены)». |
| **Токены** | Внутренняя модель (баланс генераций в legacy-контексте). В пользовательском UI не использовать; показывать «осталось фото», «пакет». |
| **Grant pack** | Действие админа: выдать пакет пользователю вручную. |
| **Модератор** | Флаг пользователя: без лимитов free/copy и rate limit. |
