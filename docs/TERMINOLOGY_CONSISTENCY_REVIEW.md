## 5. Unified recommended terminology

**Бренд:** NeoBanana  

**Продукт (пользовательский UI):**  
- Покупка: пакет, купить пакет, пакеты.  
- Результат и лимиты: фото, осталось фото, N фото.  
- Качество: 4K, без водяного знака, с водяным знаком.  
- Кнопки: «🛒 Купить пакет», «📋 В меню», «Осталось фото: N».  

**Маркетинг (промо, реферал):**  
- Допустимы: фотосессия, новый образ, бонус на фотосессию.  
- Не смешивать в одном блоке с «пакет» в смысле «единица покупки».  

**Админка (внутренняя):**  
- Заголовки: «Пакеты», «Пакеты (цены)».  
- Русские подсказки: водяной знак, без водяного знака.  
- Внутренние термины (Job, Take, токены) — без изменений, но в подписях для пользователей везде «фото», не «снимки».  

**Регистр обращения:**  
- Выбрать один: «вы» (Выберите, Купите, Отправьте) или «ты» (Выбери, Купи, Отправь) и провести по всем пользовательским экранам.

## 6. What must be renamed or rewritten

**Обязательно (перед продом):**  
1. **main.py:6171** — заменить «Нет активной фотосессии» на «Нет активного пакета.» (или оставить только «Купите пакет, чтобы начать!»).  
2. **admin-frontend PacksPage.tsx** — заголовок «Пакеты генераций» → «Пакеты» (или «Пакеты (цены)»).  
3. **PacksPage.tsx** — плейсхолдеры «5 фото без watermark» → «5 фото без водяного знака»; в ECONOMY_HELP «снять watermark» → «снять водяной знак».  

**Желательно:**  
4. Определить единый регистр (ты/вы) и пройти по приветствию, профилю, подсказкам, ошибкам.  
5. **defaults.py** — в help пункт 4: «баланс и статистика» → «остаток фото и статистика» или «пакет и статистика».  
6. **admin.py** — fallback watermark_text_effective: «NanoBanan Preview» → «NeoBanana Preview» для единого бренда.  

**Опционально:**  
7. Реферальная кнопка «Мой баланс» → «Бонусы 4K» или «Мой бонус».  
8. «Ваша сессия» в статусе → «Ваш пакет» при желании максимально убрать слово «сессия» из UI.

## 7. Final overall assessment

Терминология после унификации NeoBanana в целом приведена к пакет/фото/4K/водяной знак, но остаются:

- Один явный провал в продукте: **«Нет активной фотосессии»** в session_status (запрещённый термин в транзакционном UI).  
- В админке: **«Пакеты генераций»** и **«watermark»** в русских текстах — расхождение с глоссарием.  
- Смешение **ты/вы** и частичные остатки «баланс»/«сессия» без решения по единому тону и именованию.

**Итог:** исправить пункты 1–3 из раздела 6 обязательно; затем принять решение по единому обращению (ты/вы) и при необходимости пройти остальные пункты. После правок 1–3 продукт по терминологии можно считать согласованным с глоссарием NeoBanana для продакшена.
