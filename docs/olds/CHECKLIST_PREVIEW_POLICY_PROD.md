# Чеклист готовности к проду: политика превью и стабильность

Перед выкатом убедиться, что всё отмечено.

---

## Код и исправления

- [ ] **Миграция 063** применена на прод БД (`preview_format`, `preview_quality`, `job_preview_max_dim`, `watermark_use_contrast` в `app_settings`).
- [ ] **Утечка FD в send_media_group** — закрытие файлов в `finally` (уже в коде).
- [ ] **watermark_opacity** — валидация 0–255 при сохранении в API.
- [ ] **watermark_tile_spacing** — валидация 0–2000 при сохранении.
- [ ] **Один вариант Take** — при 1 успешном варианте используется `send_photo`, не `send_media_group`.
- [ ] **send_media_group** — проверка 2–10 элементов до открытия файлов.
- [ ] **get_or_create (app_settings)** — обработка `IntegrityError` при конкурентной вставке.
- [ ] **prepare_delivery** — превью строится от `original_path` (копия), не от raw.

## Конфиг и окружение

- [ ] В `.env` (или конфиге) задан `WATERMARK_TEXT`, если нужен fallback при пустом значении в БД.

- [ ] Достаточно места на диске для превью (рост при активной генерации).

## Админка

- [ ] Страница «Политика превью» открывается, GET `/admin/settings/preview-policy` возвращает данные.
- [ ] Сохранение настроек (PUT) работает; при невалидных значениях (формат не webp/jpeg, max_dim вне 100–4096) приходит 400 с понятным текстом.
- [ ] После сохранения новые генерации (Take/Job) используют новые формат и качество превью.

---

## Сценарии (smoke)

- [ ] **Take: 3 варианта** — генерация завершается, в чат приходит альбом из 3 фото, кнопки выбора варианта работают.
- [ ] **Take: 1 успешный вариант** (два упали) — в чат приходит одно фото через `send_photo`, без 400 от Telegram.
- [ ] **Job с превью** — превью (webp или jpeg по настройке) отправляется в Telegram, кнопки разблокировки работают, после оплаты отдаётся оригинал.
- [ ] **Старые Take/Job** — записи с путями к превью в .png по-прежнему открываются и отправляются (MIME по расширению).
- [ ] **Избранное и HD** — выбор варианта → избранное → запрос HD использует `original_path`, доставка 4K работает.
